## 🎯 เกี่ยวกับรายวิชา
เจาะลึกการทำงานของคอมพิวเตอร์ในระดับล่าง (Low-level) ผ่านภาษา **C** เพื่อให้เข้าใจโครงสร้างหน่วยความจำ, โปรเซสเซอร์, และการจัดการทรัพยากร ซึ่งเป็นรากฐานสำคัญของระบบคอมพิวเตอร์

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **Memory Management:** การจองและจัดการหน่วยความจำ การใช้งาน **Pointers** และ Array
- **Compilation Process:** ขั้นตอนจาก Source Code สู่ Machine Code (Preprocessing -> Compilation -> Assembly -> Linking)
- **Algorithmic Logic:** การเขียน Logic พื้นฐานอย่างเข้มข้น (e.g., Pyramid Patterns)

## 📂 โครงสร้างภายใน (Repository Structure)
- **`lab/`**: รวบรวม Source Code ภาษา C จากแบบฝึกหัดต่างๆ
    - `Pyramid/`: อัลกอริทึมพิมพ์รูปทรงพีระมิด
    - `Find Area/`: โปรแกรมคำนวณพื้นที่
    - `Change Values/`: การสลับค่าตัวแปร

## 🛠️ เครื่องมือที่ใช้
- **Language:** C
- **Compiler:** GCC (MinGW)
