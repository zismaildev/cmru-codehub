# Lab 1: Similarity & Distance Measures

## 🎯 วัตถุประสงค์ (Objective)
ทำความเข้าใจและฝึกเขียนโค้ดเพื่อวัด "ความเหมือน" หรือ "ความต่าง" ของข้อมูล (Data Points) ซึ่งเป็นพื้นฐานสำคัญของ Data Mining หลายๆ อัลกอริทึม (เช่น K-Nearest Neighbors, K-Means Clustering)

## 📝 เนื้อหาใน Lab
1. **Euclidean Distance**: วัดระยะกระจัด (เส้นตรง) ระหว่าง 2 จุด
2. **Manhattan Distance**: วัดระยะทางแบบเดินตามบล็อกถนน (ผลรวมความต่างของแต่ละแกน)
3. **Cosine Similarity**: วัดมุมระหว่างเวกเตอร์ (ยิ่งมุมแคบ = ยิ่งเหมือนกัน) เหมาะสำหรับ Text Data
