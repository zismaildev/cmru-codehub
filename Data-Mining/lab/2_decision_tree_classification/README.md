# Lab 2: Decision Tree Classification

## 🎯 วัตถุประสงค์ (Objective)
ฝึกสร้างแบบจำลองการเรียนรู้แบบมีผู้สอน (Supervised Learning) ประเภท Classification โดยใช้อัลกอริทึม **Decision Tree (ต้นไม้ตัดสินใจ)** เพื่อแยกแยะคลาสของข้อมูล

## 📝 เนื้อหาใน Lab
1. **Data Loading**: โหลดข้อมูลตัวอย่างจาก Scikit-Learn
2. **Train/Test Split**: แบ่งข้อมูลเป็นชุดเรียนรู้ (Training Set) และชุดทดสอบ (Test Set)
3. **Model Training**: สร้างโมเดล `DecisionTreeClassifier` และฝึกสอนด้วยข้อมูล Train
4. **Prediction**: นำโมเดลไปทำนายผลกับข้อมูล Test
5. **Tree Visualization**: (อาจมี) การพล็อตโครงสร้างต้นไม้เพื่อให้เห็นเงื่อนไขที่โมเดลใช้ตัดสินใจ

## 🚀 วิธีการรัน (How to Run)
```bash
python 2_decision_tree_classification.py
```
