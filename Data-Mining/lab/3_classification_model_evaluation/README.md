# Lab 3: Classification Model Evaluation

## 🎯 วัตถุประสงค์ (Objective)
เรียนรู้วิธีการ "วัดผล" โมเดล Classification ว่าทำงานได้ดีแค่ไหน การดูแค่ค่าความแม่นยำ (Accuracy) อย่างเดียวอาจไม่เพียงพอในโลกความเป็นจริง

## 📝 เนื้อหาใน Lab
1. **Confusion Matrix**: ตารางแสดงผลการทำนาย (True Positive, False Positive, True Negative, False Negative)
2. **Accuracy**: สัดส่วนที่ทายถูกทั้งหมด
3. **Precision**: ทายว่าเป็น Positive แล้วถูกจริงๆ กี่เปอร์เซ็นต์
4. **Recall (Sensitivity)**: ของที่เป็น Positive จริงๆ โมเดลหาเจออเจอกี่เปอร์เซ็นต์
5. **F1-Score**: ค่าเฉลี่ยฮาร์โมนิกของ Precision และ Recall (ใช้ดูภาพรวมเมื่อข้อมูลไม่สมดุล)
