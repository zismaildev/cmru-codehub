## 🎯 เกี่ยวกับรายวิชา
เรียนรู้กระบวนการตรวจสอบคุณภาพซอฟต์แวร์ **(Quality Assurance)** ทั้งแบบ Manual และ Automated Testing เพื่อหาข้อผิดพลาด (Bugs) และยืนยันว่าซอฟต์แวร์ทำงานได้ถูกต้องตามความต้องการ

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **Testing Levels:** Unit Testing, Integration Testing, System Testing
- **Test Case Design:** การเขียน Test Scenario และ Test Case ที่ครอบคลุม
- **Automated Testing:** การใช้เครื่องมือเพื่อรันการทดสอบอัตโนมัติ ลดเวลาและเพิ่มความแม่นยำ

## 📂 โครงสร้างภายใน (Repository Structure)
- **`mini-project/`**:
    - `katalon-automation/`: โปรเจกต์ทดสอบระบบอัตโนมัติด้วย Katalon Studio

## 🛠️ เครื่องมือที่ใช้
- **Tools:** Robot Framework / Katalon Studio / Selenium

---

## 🚀 วิธีการรัน (How to Run)
1. เปิดโปรเจกต์ด้วย **Katalon Studio** หรือเครื่องมือที่เกี่ยวข้อง
2. เลือก Test Suite ที่ต้องการ
3. กด **Run** เพื่อดูผลการทดสอบ
