## 🎯 เกี่ยวกับรายวิชา
รายวิชานี้เน้นการเขียนโปรแกรมเชิงวัตถุ **(Object-Oriented Programming - OOP)** ด้วยภาษา **Java** เพื่อแก้ปัญหาความซับซ้อนของการเขียนโค้ดแบบ Procedural โดยเรียนรู้วิธีการมอง Code ให้เป็น "วัตถุ" ที่ทำงานร่วมกัน

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **The 4 Pillars of OOP:**
  1.  **Encapsulation**: การซ่อนข้อมูลและปกป้อง State ภายใน
  2.  **Abstraction**: การลดความซับซ้อนโดยโชว์เฉพาะสิ่งที่จำเป็น
  3.  **Inheritance**: การสืบทอดคุณสมบัติเพื่อ Reusability
  4.  **Polymorphism**: การทำงานหลายรูปแบบใน Method เดียวกัน
- **Memory Management:** ความเข้าใจเรื่อง Stack vs Heap
