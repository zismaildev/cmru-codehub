## 🎯 เกี่ยวกับรายวิชา
พื้นฐานวิทยาการข้อมูล เรียนรู้กระบวนการ **Data Science Pipeline** ตั้งแต่การรวบรวมข้อมูล (Data Collection), การทำความสะอาดข้อมูล (Cleaning), การวิเคราะห์ (Analysis) ไปจนถึงการนำเสนอข้อมูลด้วยภาพ (Visualization)

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **Data Manipulation:** การใช้ Library **Pandas** และ **NumPy** ในการจัดการชุดข้อมูลขนาดใหญ่
- **Data Visualization:** การสร้างกราฟด้วย **Matplotlib** และ **Seaborn** เพื่อสื่อความหมาย
- **Web Scraping:** การดึงข้อมูลจากเว็บไซต์มาวิเคราะห์
- **Machine Learning Basics:** แนวคิดเบื้องต้นเกี่ยวกับการสร้าง Model ทำนายผล

## 📂 โครงสร้างภายใน (Repository Structure)
- **`lab/`**: ไฟล์ Jupyter Notebook (`.ipynb`) จากการทำ Lab รายสัปดาห์ (Lab 1-17) และแบบฝึกหัด Colab
- **`mini-project/`**:
    - `Web Scraping Group`: โปรเจกต์กลุ่มเกี่ยวกับการดึงข้อมูลเว็บ

## 🛠️ เครื่องมือที่ใช้
- **Language:** Python
- **Environment:** Google Colab / Jupyter Notebook
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
