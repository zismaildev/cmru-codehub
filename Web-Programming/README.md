## 🎯 เกี่ยวกับรายวิชา
ศึกษาหลักการทำงานของ **Web Application** ตั้งแต่ฝั่ง Client (Frontend) ไปจนถึง Server (Backend) และ Database โดยเน้นการสร้างเว็บที่สามารถโต้ตอบกับผู้ใช้และจัดการข้อมูลได้จริง

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **Frontend Basics:** โครงสร้าง HTML5, การตกแต่งด้วย CSS3, และ Interactive UI ด้วย JavaScript
- **Backend Logic:** การเขียน PHP เพื่อประมวลผลข้อมูลจาก Form
- **Database Connection:** การเชื่อมต่อ MySQL และทำ CRUD Operations (Create, Read, Update, Delete)
- **Session & Cookies:** การจัดการสถานะของผู้ใช้ (Stateless vs Stateful)

## 📂 โครงสร้างภายใน (Repository Structure)
- **`lab/`**:
    - `static-web/`: โปรเจกต์ HTML/CSS พื้นฐาน
    - `database-connection/`: ตัวอย่างการเชื่อมต่อฐานข้อมูล
    - `legacy-php/`: โค้ด PHP แบบดั้งเดิม
- **`mini-project/`**:
    - `ecommerce-system/`: ระบบร้านค้าออนไลน์แบบครบวงจร (Final Project)

## 🛠️ เครื่องมือที่ใช้
- **Stack:** HTML, CSS, PHP, MySQL
- **Server:** XAMPP / Apache

---

## 🚀 วิธีการรัน (How to Run)
1. ติดตั้ง **XAMPP** และ Start Apache + MySQL
2. นำโฟลเดอร์โปรเจกต์ไปวางที่ `htdocs`
3. เปิด Browser เข้าไปที่ `http://localhost/<folder-name>`
