## 🎯 เกี่ยวกับรายวิชา
ศึกษาการพัฒนาแอปพลิเคชันบนอุปกรณ์เคลื่อนที่ด้วย **Flutter Framework** ซึ่งช่วยให้สามารถเขียนโค้ดชุดเดียวแต่รันได้ทั้งบน iOS และ Android (Cross-platform) เน้นการออกแบบ UI ที่สวยงามและการจัดการ State ที่มีประสิทธิภาพ

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **Widget Tree:** ความเข้าใจโครงสร้างของ Flutter ที่ทุกอย่างเป็น Widget
- **State Management:** การจัดการข้อมูลในแอป (Stateless vs Stateful, Provider)
- **User Interface:** การออกแบบ Layout ที่รองรับหน้าจอหลายขนาด (Responsive)
- **Asynchronous Programming:** การดึงข้อมูลจาก API ด้วย `Future` และ `Stream`

## 📂 โครงสร้างภายใน (Repository Structure)
- **`mini-project/`**:
    - `flutter-app/`: ซอร์สโค้ดของแอปพลิเคชันหลักที่พัฒนาในวิชา

## 🛠️ เครื่องมือที่ใช้
- **Framework:** Flutter SDK
- **Language:** Dart
- **IDE:** VS Code / Android Studio
