## 🎯 เกี่ยวกับรายวิชา

รายวิชานี้เป็น**จุดเริ่มต้นของการเขียนโปรแกรม** สำหรับนักศึกษาวิทยาการคอมพิวเตร์ทุกคน มุ่งเน้นการพัฒนา **Computational Thinking** หรือกระบวนการคิดเชิงคำนวณ เพื่อฝึกให้สามารถแก้ปัญหาได้อย่างเป็นระบบ

ใช้ภาษา **Python** เป็นสื่อในการเรียนรู้ เนื่องจาก:
- **Syntax กระชับ**: อ่านเข้าใจง่าย คล้ายภาษาอังกฤษทั่วไป
- **ไม่ซับซ้อน**: ไม่ต้องกังวลเรื่อง Type Declaration หรือ Memory Management
- **เหมาะกับผู้เริ่มต้น**: สามารถมองเห็นผลลัพธ์ได้ทันที

### เนื้อหาหลักที่เรียน
1. **Basic Syntax** - Variables, Data Types, Operators
2. **Control Flow** - Conditional Statements (if-else), Loops (for, while)
3. **Data Structures** - List, Dictionary, Tuple, Set
4. **Functions** - การสร้างฟังก์ชันและการเรียกใช้งาน
5. **File I/O** - การอ่านและเขียนไฟล์
6. **Error Handling** - การจัดการข้อผิดพลาดด้วย try-except
7. **Algorithm Design** - การออกแบบขั้นตอนการแก้ปัญหา

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)

### 1. Computational Thinking
- **Decomposition**: แบ่งปัญหาใหญ่ให้เป็นปัญหาย่อยๆ
- **Pattern Recognition**: มองเห็นรูปแบบที่ซ้ำกันในปัญหา
- **Abstraction**: แยกส่วนสำคัญออกจากรายละเอียดที่ไม่จำเป็น
- **Algorithm Design**: เขียนขั้นตอนการแก้ปัญหาอย่างเป็นระบบ

### 2. Control Flow Mastery
- **Conditional Statements**: การใช้ `if`, `elif`, `else` เพื่อตัดสินใจ
- **For Loops**: การวนซ้ำจำนวนที่แน่นอน (Iteration)
- **While Loops**: การวนซ้ำตามเงื่อนไข (Condition-based)
- **Nested Loops**: การใช้ Loop ซ้อน Loop เพื่อแก้ปัญหาซับซ้อน

### 3. Data Structures in Python
- **Lists**: เก็บข้อมูลเป็นลำดับ (Ordered Collection) - `[1, 2, 3]`
- **Dictionaries**: เก็บข้อมูลแบบ Key-Value - `{"name": "John", "age": 20}`
- **Tuples**: ข้อมูลที่ไม่เปลี่ยนแปลง (Immutable) - `(x, y)`
- **Sets**: ข้อมูลที่ไม่ซ้ำกัน (Unique Collection) - `{1, 2, 3}`

### 4. Functions & Modular Programming
- **Function Definition**: การสร้างฟังก์ชันด้วย `def`
- **Parameters & Arguments**: การส่งค่าเข้าฟังก์ชัน
- **Return Values**: การส่งค่ากลับจากฟังก์ชัน
- **Scope**: ขอบเขตของตัวแปร (Local vs Global)

### 5. Problem Solving Skills
- **Input Validation**: การตรวจสอบข้อมูลที่ผู้ใช้ป้อน
- **Edge Cases**: การจัดการกรณีพิเศษ (เช่น input เป็น 0, ติดลบ)
- **Debugging**: การหาและแก้ไข Bug ในโค้ด
- **Code Optimization**: การปรับปรุงโค้ดให้ทำงานเร็วขึ้น

## 🚀 วิธีการรัน (How to Run)

### 📋 Prerequisites
1. ติดตั้ง Python:
   - Windows: Download จาก [python.org](https://www.python.org/)
   - Mac: มีมาให้แล้ว หรือติดตั้งผ่าน Homebrew
   - Linux: `sudo apt install python3`

2. ตรวจสอบว่าติดตั้งสำเร็จ:
   ```bash
   python --version
   # หรือ
   python3 --version
   ```

### ▶️ รันโปรแกรม
1. เปิด Terminal/Command Prompt
2. ไปที่โฟลเดอร์ที่มีไฟล์ `.py`
3. รันด้วยคำสั่ง:
   ```bash
   python filename.py
   # หรือ
   python3 filename.py
   ```

**ตัวอย่าง:**
```bash
cd Computer-Programming/assignment/assignment\ 1/
python main.py
```

### 🔧 Troubleshooting
- **`python` ไม่ทำงาน**: ลอง `python3` แทน (บน Mac/Linux)
- **ไม่เจอคำสั่ง**: ตรวจสอบว่าเพิ่ม Python ลใน PATH แล้ว
- **Error แปลกๆ**: ตรวจสอบ Indentation (Python ใช้เว้นวรรคบังคับ)
