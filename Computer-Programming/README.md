## 🎯 เกี่ยวกับรายวิชา
รายวิชานี้มุ่งเน้นการปูพื้นฐาน **Computational Thinking** หรือกระบวนการคิดเชิงคำนวณ ผ่านภาษา **Python** ซึ่งเป็นภาษาที่มี Syntax กระชับและเข้าใจง่าย เหมาะสำหรับการฝึกเขียน Logic, Algorithm และการแก้ปัญหาเบื้องต้น

## 🔑 สิ่งที่เรียนรู้ (Key Learnings)
- **Control Flow:** การใช้ `if-else`, loops (`for`, `while`) ในการควบคุมการทำงานของโปรแกรม
- **Data Structures:** การจัดการข้อมูลด้วย List, Dictionary, Set และ Tuple
- **Functions:** การเขียนฟังก์ชันเพื่อลดความซ้ำซ้อน (DRY Principle) และ Modular Programming
- **Error Handling:** การจัดการข้อผิดพลาดด้วย `try-except`

## 📂 โครงสร้างภายใน (Repository Structure)
- **`assignment/`**: แบบฝึกหัดท้ายบทและงานที่ได้รับมอบหมาย (e.g., Logic Puzzles, Basic Scripts)
- **`lab/`**: ไฟล์ Lab ปฏิบัติการในห้องเรียน และแบบทดสอบ (Practical Tests)

## 🛠️ เครื่องมือที่ใช้
- **Language:** Python 3.x
- **IDE:** VS Code / PyCharm / Jupyter Notebook
